# Instrucciones para ejecutar el Backend

Este backend está construido con Node.js y Express.

## Instalación

```bash
npm install
```

## Variables de entorno

Crea un archivo `.env` con:

```
PORT=3001
CORS_ORIGIN=http://localhost:3000
```

## Ejecución en desarrollo

```bash
npm run dev
```

## Tests

```bash
npm test
```
